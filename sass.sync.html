Sass.js Sync Console

This document loads the synchronous sass.js. Please open your browser's DevTools to use the API.

Try the following example:

Sass.writeFile('testfile.scss', '@import "sub/deeptest";\n.testfile { content: "loaded"; }');
Sass.writeFile('sub/deeptest.scss', '.deeptest { content: "loaded"; }');
Sass.options({ style: Sass.style.expanded });
var result = Sass.compile('@import "testfile";');
console.log("compiled", result.text);
